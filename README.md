# Simon Say Game 🎮

A simple and interactive Simon Says game built using HTML, CSS, and JavaScript.

## 🚀 How to Play
- Press any key or click to start the game
- Watch the flashing color sequence
- Repeat the same sequence by clicking the buttons
- Sequence gets longer with each level
- Game ends if you press the wrong button

## ✨ Features
- Color flash animations
- Increasing difficulty levels
- Score display on game over
- Responsive design (mobile, tablet, desktop)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Files
- `index.html`
- `style.css`
- `app.js`
